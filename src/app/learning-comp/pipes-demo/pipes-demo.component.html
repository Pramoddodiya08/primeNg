Date Pipe:

Transforms a Date object into a readable string format.

Full Date: {{ currentDate | date : "fullDate" }}

Example output: "Thursday, November 14, 2024"

Short Time: {{ currentDate | date : "shortTime" }}

Example output: "3:45 PM"

UpperCase Pipe:

Transforms all characters of the text to uppercase.

{{ sampleText | uppercase }}

Example output: "LEARNING ANGULAR PIPES!"

LowerCase Pipe:

Transforms all characters of the text to lowercase.

{{ sampleText | lowercase }}

Example output: "learning angular pipes!"

Currency Pipe:

Transforms a number to a currency format based on locale.

USD: {{ price | currency : "USD" }}

Example output: "$12,345.68"

INR: {{ price | currency : "INR" : "symbol-narrow" }}

Example output: "â‚¹12,345.68"

Decimal Pipe:

Formats a number according to a given decimal pattern.

{{ price | number : "1.2-2" }}

Example output: "12,345.68"

Percent Pipe:

Transforms a decimal number into a percentage format.

{{ percentage | percent }}

Example output: "85%"

JSON Pipe:

Transforms a JavaScript object into a JSON string.

{{ jsonObject | json }}

Example output: "name":"Angular","version":16,"standalone":true
